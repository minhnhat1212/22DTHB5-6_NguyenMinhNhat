AJAX và Fetch API: Giao tiếp bất đồng bộ trong JavaScript

JavaScript 10/12/2024 3 phút đọc

AJAX (Asynchronous JavaScript and XML) và Fetch API là hai công nghệ quan trọng cho phép JavaScript giao tiếp với server mà không cần reload trang.

1. Nó là gì?

AJAX là một kỹ thuật cho phép web applications gửi và nhận dữ liệu từ server một cách bất đồng bộ mà không cần reload toàn bộ trang.

Fetch API là một interface hiện đại và đơn giản hơn để thực hiện HTTP requests, sử dụng Promises thay vì callbacks.

2. Nó giúp ích cho tôi ra sao khi sử dụng?

  • Tăng tốc độ ứng dụng: Không cần reload trang, chỉ cập nhật phần cần thiết
  • Tiết kiệm băng thông: Chỉ tải dữ liệu cần thiết thay vì toàn bộ trang
  • Cải thiện trải nghiệm người dùng: Giao diện mượt mà, không bị gián đoạn
  • Xây dựng ứng dụng real-time: Có thể cập nhật dữ liệu liên tục

3. Cách sử dụng

3.1. XMLHttpRequest (AJAX)

// GET Request
function getDataWithXHR() {
    const xhr = new XMLHttpRequest();
    xhr.open('GET', 'https://jsonplaceholder.typicode.com/posts/1', true);
    
    xhr.onreadystatechange = function() {
        if (xhr.readyState === 4 && xhr.status === 200) {
            const data = JSON.parse(xhr.responseText);
            console.log('Dữ liệu:', data);
        }
    };
    
    xhr.send();
}

// POST Request
function postDataWithXHR() {
    const xhr = new XMLHttpRequest();
    const data = { title: 'Bài viết mới', body: 'Nội dung', userId: 1 };
    
    xhr.open('POST', 'https://jsonplaceholder.typicode.com/posts', true);
    xhr.setRequestHeader('Content-Type', 'application/json');
    
    xhr.onreadystatechange = function() {
        if (xhr.readyState === 4 && xhr.status === 201) {
            console.log('Đã tạo:', JSON.parse(xhr.responseText));
        }
    };
    
    xhr.send(JSON.stringify(data));
}

3.2. Fetch API (Hiện đại)

// GET Request với async/await
async function getDataWithFetch() {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/posts/1');
        const data = await response.json();
        console.log('Dữ liệu:', data);
    } catch (error) {
        console.error('Lỗi:', error);
    }
}

// POST Request với async/await
async function postDataWithFetch() {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/posts', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({
                title: 'Bài viết mới',
                body: 'Nội dung',
                userId: 1
            })
        });
        
        const result = await response.json();
        console.log('Đã tạo:', result);
    } catch (error) {
        console.error('Lỗi:', error);
    }
}

4. Kết luận

Khuyến nghị: Sử dụng Fetch API cho các dự án mới vì syntax đơn giản và hỗ trợ Promise tốt hơn XMLHttpRequest.
