TypeScript: JavaScript với Type Safety

Lợi ích của types, cách định nghĩa interface/type, generics và thiết lập project cơ bản.

Vì sao cần TypeScript

Giảm bug runtime, auto-completion tốt hơn, refactor an toàn, tài liệu hóa code qua types.

Các khối xây dựng

  • Type/Interface cho cấu trúc dữ liệu
  • Generics cho tính tái sử dụng
  • Union/Intersection types cho mô tả linh hoạt

Interfaces và Generics

interface ApiResponse {
  data: T;
  error?: string;
}

type User = { id: number; name: string };

function wrap(data: T): ApiResponse {
  return { data };
}

const res = wrap({ id: 1, name: 'Alice' });

Narrowing & Guards

Type guards

type Cat = { meow: () => void };
type Dog = { bark: () => void };
type Pet = Cat | Dog;

function isCat(p: Pet): p is Cat { return 'meow' in p; }

function speak(p: Pet) {
  if (isCat(p)) p.meow(); else p.bark();
}
