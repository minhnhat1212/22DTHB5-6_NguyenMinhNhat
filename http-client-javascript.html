HTTP Client trong JavaScript: Từ XMLHttpRequest đến Fetch API

JavaScript 18/12/2024 4 phút đọc

Mục lục

  • 1. XMLHttpRequest - Phương pháp truyền thống
  • 2. Fetch API - Phương pháp hiện đại
  • 3. Axios - Thư viện HTTP Client
  • 4. Error Handling
  • 5. Best Practices

HTTP Client là một phần quan trọng trong lập trình web hiện đại, cho phép JavaScript thực hiện các HTTP requests để giao tiếp với server.

1. Nó là gì?

XMLHttpRequest là API đầu tiên cho phép JavaScript thực hiện HTTP requests, mặc dù đã cũ nhưng vẫn được hỗ trợ rộng rãi.

Fetch API là interface hiện đại và dễ sử dụng hơn để thực hiện HTTP requests, sử dụng Promises và async/await.

Axios là một thư viện HTTP client phổ biến, cung cấp nhiều tính năng tiện ích như request/response interceptors và error handling tốt hơn.

2. Nó giúp ích cho tôi ra sao khi sử dụng?

  • Giao tiếp với API: Lấy và gửi dữ liệu từ/đến server
  • Xây dựng ứng dụng SPA: Tạo Single Page Application không cần reload
  • Tích hợp với backend: Kết nối frontend với các service backend
  • Xử lý dữ liệu real-time: Cập nhật dữ liệu liên tục từ server
  • Upload/Download files: Gửi và nhận files qua HTTP

3. Cách sử dụng

3.1. XMLHttpRequest

// GET Request
function getDataWithXHR() {
    const xhr = new XMLHttpRequest();
    xhr.open('GET', 'https://jsonplaceholder.typicode.com/posts/1', true);
    
    xhr.onreadystatechange = function() {
        if (xhr.readyState === 4 && xhr.status === 200) {
            const data = JSON.parse(xhr.responseText);
            console.log('Dữ liệu:', data);
        }
    };
    
    xhr.send();
}

// POST Request
function postDataWithXHR() {
    const xhr = new XMLHttpRequest();
    const data = { title: 'Bài viết mới', body: 'Nội dung', userId: 1 };
    
    xhr.open('POST', 'https://jsonplaceholder.typicode.com/posts', true);
    xhr.setRequestHeader('Content-Type', 'application/json');
    
    xhr.onreadystatechange = function() {
        if (xhr.readyState === 4 && xhr.status === 201) {
            console.log('Đã tạo:', JSON.parse(xhr.responseText));
        }
    };
    
    xhr.send(JSON.stringify(data));
}

3.2. Fetch API

// GET Request
async function getDataWithFetch() {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/posts/1');
        const data = await response.json();
        console.log('Dữ liệu:', data);
    } catch (error) {
        console.error('Lỗi:', error);
    }
}

// POST Request
async function postDataWithFetch() {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/posts', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({
                title: 'Bài viết mới',
                body: 'Nội dung',
                userId: 1
            })
        });
        
        const result = await response.json();
        console.log('Đã tạo:', result);
    } catch (error) {
        console.error('Lỗi:', error);
    }
}

3.3. Axios

// Cài đặt: npm install axios
import axios from 'axios';

// GET Request
async function getDataWithAxios() {
    try {
        const response = await axios.get('https://jsonplaceholder.typicode.com/posts/1');
        console.log('Dữ liệu:', response.data);
    } catch (error) {
        console.error('Lỗi:', error.message);
    }
}

// POST Request
async function postDataWithAxios() {
    try {
        const response = await axios.post('https://jsonplaceholder.typicode.com/posts', {
            title: 'Bài viết mới',
            body: 'Nội dung',
            userId: 1
        });
        console.log('Đã tạo:', response.data);
    } catch (error) {
        console.error('Lỗi:', error.message);
    }
}
