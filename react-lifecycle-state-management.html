React.js: Component Lifecycle và State Management

Hiểu vòng đời component, quản lý state với useState/useEffect và tối ưu hiệu năng.

Lifecycle hiện đại

Với function components, lifecycle thể hiện qua useEffect (mount/update/unmount). Tránh lạm dụng nhiều effects nhỏ gây re-render.

Quản lý state

  • State local: useState, useReducer
  • State chia sẻ: Context API hoặc libraries (Zustand, Redux)
  • Server state: React Query giúp caching, syncing

useState và useEffect ví dụ

import { useEffect, useState } from 'react';

export default function Counter() {
  const [count, setCount] = useState(0);
  useEffect(() => {
    document.title = `Count: ${count}`;
  }, [count]);
  return (
    <button onClick={() => setCount(c => c + 1)}>Clicked {count} times</button>
  );
}

Tối ưu

  • memo/useMemo/useCallback để giảm render không cần thiết
  • Code splitting và lazy loading cho routes/components nặng

Mẫu kiến trúc state

  • Smart/Dumb components: container xử lý logic, presentational hiển thị
  • Query state (server) tách khỏi UI state (client)

Context nhẹ cho theme

const ThemeContext = createContext('light');
function App() {
  const [theme, setTheme] = useState('light');
  return (
    <ThemeContext.Provider value={{ theme, setTheme }}>
      <Page />
    </ThemeContext.Provider>
  );
}
