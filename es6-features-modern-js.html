ES6+ Features: Những tính năng JavaScript hiện đại bạn cần biết

Tổng hợp ngắn gọn nhưng thực dụng các tính năng ES6+: let/const, arrow function, destructuring, spread/rest, modules và async/await.

Cú pháp hiện đại giúp code gọn hơn

  • let/const: tránh hoisting khó hiểu của var, tăng tính an toàn
  • Arrow functions: lexical this, code ngắn
  • Destructuring: lấy dữ liệu từ object/array nhanh
  • Spread/Rest: sao chép/hợp nhất và nhận tham số linh hoạt

Modules và Async/Await

Modules (import/export) giúp chia nhỏ code, dễ tái sử dụng. Async/Await làm rõ luồng bất đồng bộ, thay thế callback hell.

Async/Await ví dụ

// api.js
export async function fetchUsers() {
  const res = await fetch('https://jsonplaceholder.typicode.com/users');
  if (!res.ok) throw new Error('Network error');
  return res.json();
}

// main.js
import { fetchUsers } from './api.js';

(async () => {
  try {
    const users = await fetchUsers();
    console.log(users.map(u => u.name));
  } catch (e) {
    console.error(e.message);
  }
})();

Best practices

  • Dùng const mặc định, chuyển sang let khi cần reassign
  • Ưu tiên arrow functions cho callbacks
  • Tránh lồng await trong vòng lặp, dùng Promise.all khi phù hợp

Pitfalls thường gặp

  • this trong arrow function là lexical, không dùng được làm method cần bind
  • Destructuring với default values có thể che lỗi null/undefined
  • Top-level await chỉ khả dụng trong ESM, chú ý môi trường build

Promise.all tối ưu vòng lặp

const urls = ['a.json','b.json','c.json'];
const results = await Promise.all(urls.map(u => fetch(u).then(r => r.json())));
console.log(results);

Hình minh họa / Sơ đồ

Module dependency (giản lược)

[main.js] → imports → [api.js]
[main.js] → imports → [utils.js]
