Tối ưu hóa Performance cho Web Application

Kỹ thuật tối ưu tải trang và tương tác: lazy-loading, code-splitting, caching và quan trắc hiệu năng.

Tối ưu tải trang

  • Giảm kích thước bundle, tree-shaking
  • Lazy load images/components, prefetch routes
  • Sử dụng CDN và HTTP/2

Đo lường và quan trắc

  • Lighthouse, WebPageTest, Core Web Vitals (LCP, CLS, INP)
  • Performance API đo thời gian thực trên client

Đo LCP đơn giản

new PerformanceObserver((list) => {
  for (const entry of list.getEntries()) {
    console.log('LCP:', entry.startTime);
  }
}).observe({ type: 'largest-contentful-paint', buffered: true });

Chiến lược cache

HTTP caching, Service Worker cho offline-first, cache busting bằng hashed filenames.

Service Worker cache cơ bản

const CACHE = 'app-v1';
self.addEventListener('install', e => {
  e.waitUntil(caches.open(CACHE).then(c => c.addAll(['/','/styles.css','/script.js'])));
});
self.addEventListener('fetch', e => {
  e.respondWith(caches.match(e.request).then(r => r || fetch(e.request)));
});
